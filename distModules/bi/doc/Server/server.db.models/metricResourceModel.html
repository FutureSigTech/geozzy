• metricResourceModel.js

  • ¶

    Mongoose model for resource metrics

  • ¶
    • mongoose: Mongoose instance
    • Schema: Mongoose Schema instance
    var mongoose = require('mongoose'),
    Schema = mongoose.Schema;
  • ¶

    Schema fields:

    • _observation: Reference to the observation related to
    • metricTime: Date and time of the metric
    • resource: Resource object
    • event: Event object
    • section: Part of the page related to
    • seconds: Number of seconds while events were generated
    var metricResourceSchema = new Schema({
    _observation: { type: Schema.Types.ObjectId, ref: 'Observation' },
    metricTime: { type: Date, required: true },
    resource: { type: Object, required: true },
    event: { type: Object, required: true },
    section: { type: String, required: true },
    seconds: { type: Number, required: true }
});
  • ¶

    Create a new model using the schema

    var metricResourceModel = mongoose.model('MetricResource', metricResourceSchema);
  • ¶

    Exported:

    • metricResourceSchema: Schema for content metric
    • metricResourceModel: Model for content metric
    exports.metricResourceSchema = metricResourceSchema;
exports.metricResourceModel = metricResourceModel;
