  • ¶

    Mongoose model for exploration metrics

  • ¶
    • mongoose: Mongoose instance
    • Schema: Mongoose Schema instance
    var mongoose = require('mongoose'),
    Schema = mongoose.Schema;
  • ¶

    Schema fields:

    • _observation: Reference to the observation related to
    • metricTime: Date and time of the metric
    • explorer_ID: ID of the explorer
    • bounds: List of bounds: Each bound is a pair of points that represent a location
    • filters: Filters applied for this metric
    var metricExplorationSchema = new Schema({
    _observation: { type: Schema.Types.ObjectId, ref: 'Observation' },
    metricTime: { type: Date, required: true },
    explorer_ID: { type: Number, required: true },
    bounds: { type: Array, required: true},
    filters: [{ type: String, required: true }]
});
  • ¶

    Create a new model using the schema

    var metricExplorationModel = mongoose.model('MetricExploration', metricExplorationSchema);
  • ¶

    Exported:

    • metricResourceSchema: Schema for explorer metric
    • metricResourceModel: Model for explorer metric
    exports.metricExplorationSchema = metricExplorationSchema;
exports.metricExplorationModel = metricExplorationModel;
