  • ¶

    Mongoose model for observations

  • ¶
    • mongoose: Mongoose instance
    var mongoose = require('mongoose');
  • ¶

    Schema fields:

    • user_ID: User related to the observation
    • session_ID: Session related to the observation
    • device: Device object
    • observationTime: Date and time of the observation
    • language: Language used by the user
    • remoteIP: Metrics array
    • remoteLocation: IP address of the public
    var observationSchema = new mongoose.Schema({
    user_ID: { type: Number, required: true },
    session_ID: { type: String, required: true },
    device: { type: Object, required: true },
    observationTime: { type: Date, required: true },
    language: { type: String, required: true },
    metrics: { type: Array, required: true },
    remoteIP: { type: String },
    remoteLocation: { type: Object }
});
  • ¶

    Create a new model using the schema

    var observationResourceModel = mongoose.model('ObservationResource', observationSchema);
var observationExplorerModel = mongoose.model('ObservationExplorer', observationSchema);
  • ¶

    Exported:

    • observationSchema: Schema for observation
    • observationResourceModel: Model for resource observation
    • observationExplorerModel: Model for explorer observation
    exports.observationSchema = observationSchema;
exports.observationResourceModel = observationResourceModel;
exports.observationExplorerModel = observationExplorerModel;
