    Charts configuration module

  • ¶
    • constants: constants module
    var constants = require('constants');
  • ¶

    Chart configuration fields:

    • chartType: chart type ID
    • chartName: chart name
    • groupByIDs: group by IDs that make sense with each chart
    module.exports = [
    {
        chartType: constants.BAR_CHART,
        chartName: "Grafica de Barras",
        groupByIDs: [
            constants.GROUP_BY_RESOURCE_NAME,
            constants.GROUP_BY_RESOURCE_TYPE,
            constants.GROUP_BY_RESOURCE_TOPIC,
            constants.GROUP_BY_RESOURCE_TERM,
            constants.GROUP_BY_DEVICE_TYPE,
            constants.GROUP_BY_LANGUAGE,
            constants.GROUP_BY_SECTION
        ]
    },
    {
        chartType: constants.HEAT_MAP,
