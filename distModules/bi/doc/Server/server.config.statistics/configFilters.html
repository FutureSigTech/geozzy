  • ¶

    Filters configuration file

  • ¶
    • constants: constants module
    var constants = require('./constants');
  • ¶

    Filter configuration fields:

    • filterID: filter ID
    • filterType: filter type
    • filterName: filter name
    • filed: MongoDB field which the filter is applied to
    • operator: MongoDB query operator
    • value: Default filter value
    • isInteger: If true the filter value will be casted to integer
    • isDate: If true the filter value will be casted to a Date,
    • titleResource: Title that will be shown if we are dealing with a resource metric
    • titleExplorer: Title that will be shown if we are dealing with an explorer metric
