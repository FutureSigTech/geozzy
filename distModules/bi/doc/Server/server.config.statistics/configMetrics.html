    • MongoDB Aggregation Pipeline Stages
  • ¶
    • constants: constants module
    var constants = require('./constants');
  • ¶

    Metric configuration fields:

    • type: metric type
    • metricID: metric ID
    • model: MongoDB collection to be queried
    • groups: group by IDs that make sense with this metric
    • filter: filter associated
    • group: $group stage configuration
    • derivedMetricIDs: associated metric IDs for derived metrics
