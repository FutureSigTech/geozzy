    ExpressJS routes configuration module

  • ¶
    • express: web framework for NodeJS used to implement a webservice.
    • rootRouter: / requests handler
    • authRooter: / request handler used for authentication
    • observationRouter: /observation request handler
    • statisticsRouter: /statistics request handler
    • statisticsConfigRouter: /statistics/config request handler
    • configServer: server configuration module
    var express = require('express'),
