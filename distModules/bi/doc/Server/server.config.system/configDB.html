• configDB.js

  • ¶

    Database configuration module

  • ¶
    • insertRootUrl: Master Database url. User for inserting observations
    • resultRootUrl: Result Database url. User for storage statistics
    • replicaRootUrl: Replica Database url. Used for decrease load to Master.
    var insertRootUrl = 'mongodb://baleo.itg.es:27017',
    resultRootUrl = 'mongodb://dubra.itg.es:27017',
    replicaRootUrl = 'mongodb://dubra.itg.es:27018';
  • ¶

    Append Database name

    module.exports = {
    dbInsertUrl: insertRootUrl+'/geozzy',
    dbResultUrl: resultRootUrl+'/geozzy',
    dbReplicaUrl: replicaRootUrl+'/geozzy'
};
