• configServer.js

  • ¶

    Server configuration module

  • ¶
    • IP: IP address of NodeJS server
    • port: port of NodeJS server
    • insertObservationServer: Whether or not to configure the server as an insert observation server. Else it will be configured as a statistics server.
    • logEnabled: Whether or not to store server log
    • tokenExpiration: Token expiration time in minutes
    • authToken: Enable authentication with token
    • validationRequestEnabled: Whether or not to validate request
    module.exports = {
    IP: '127.0.0.1',
    port: 8080,
    insertObservationServer: false,
    logEnabled: false,
    tokenExpiration: 60,
    authToken: false,
    validationRequestEnabled: false
};
